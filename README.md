
# Agri-Assist

Agri-Assist is an AI-powered agriculture support platform that helps farmers with crop recommendations, weather updates, soil analysis, pest management, and multilingual AI assistance.

## Features
- AI Farming Assistant
- Weather Forecasting
- Crop Recommendations
- Pest & Disease Detection
- Marketplace for Agricultural Products
- Multilingual Support

## Technologies Used
- React.js
- TypeScript
- Vite
- Node.js
- Google Gemini AI

## Installation

```bash
npm install
npm run dev
